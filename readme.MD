# 🛵 Sistema de Gestión de Cadetería

Un sistema de gestión de cadetería desarrollado como aplicación de consola utilizando **C#** y **.NET**. 

Este proyecto es una reversión de un trabajo práctico universitario enfocado en aplicar conceptos de Programación Orientada a Objetos (POO), manejo de colecciones y lógica de negocio en consola.

## ⚙️ Tecnologías Utilizadas
* **Lenguaje:** C#
* **Framework:** .NET (Console Application)
* **Entorno de desarrollo:** Visual Studio / VS Code

## 📋 Características Principales
El sistema permite simular el flujo básico del trabajo de una empresa de envíos, incluyendo:
* **Gestión de Pedidos:** Alta de nuevos pedidos con la información del cliente.
* **Gestión de Cadetes:** Asignación y seguimiento de pedidos a cadetes específicos.
* **Control de Estados:** Cambio de estado de los pedidos (ej. de "Pendiente" a "Entregado").
* **Reportes:** Visualización de la cantidad de envíos realizados por cada cadete y el total a pagarles al final de la jornada.
